1. Consider the following method that is intended to return `true` if a string contains at least one vowel (a, e, i, o, u) and `false` otherwise.

```java
public static boolean hasVowel(String word) {
    char[] letters = word.toCharArray();
    for (int i = 0; i < letters.length; i++) {
        String ch = "" + letters[i];
        if (ch.equals("a") || ch.equals("e") || ch.equals("i") || 
            ch.equals("o") || ch.equals("u")) {
            return true;
        }
    }
    return false;
}
```

Which of the following method calls will return `true`?

I. `hasVowel("AEIOU")`
II. `hasVowel("rhythm")`
III. `hasVowel("Apple")`

a. III only
a. I only
a. I and III only
a. II and III only

3. Consider the following method that is intended to return the number of elements in an integer array that are strictly greater than all elements that come after them.

```java
public static int countGreaterThanRest(int[] nums) {
    int count = 0;
    for (int i = 0; i < nums.length; i++) {
        boolean isGreater = true;
        for (int j = i + 1; j < nums.length; j++) {
            if (nums[i] <= nums[j]) {
                isGreater = false;
            }
        }
        if (isGreater) {
            count++;
        }
    }
    return count;
}
```

What value is returned by the call `countGreaterThanRest(new int[] {5, 3, 4, 2, 1})`?

a. 3
a. 1
a. 2
a. 4
