1. Consider the following method that is intended to return a new string where every occurrence of a two-character pattern is replaced with a single uppercase version of the first character of that pattern.

```java
public String condensePattern(String str, String pattern)
{
    String result = "";
    int pos = 0;
    
    while (pos < str.length())
    {
        if (pos <= str.length() - pattern.length() && 
            str.substring(pos, pos + pattern.length()).equals(pattern))
        {
            result += pattern.substring(0, 1).toUpperCase();
            pos += pattern.length();
        }
        else
        {
            result += str.substring(pos, pos + 1);
            pos++;
        }
    }
    return result;
}
```

Assuming `pattern` always has length 2, which of the following method calls will execute WITHOUT throwing an exception?

a. I, II, and III: `condensePattern("ababcc", "ab")`, `condensePattern("xxyyzz", "xy")`, `condensePattern("aaa", "aa")` b. I and II only c. I and III only d. II only
